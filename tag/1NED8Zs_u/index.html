《知乎圆桌·牙齿保卫战》
2019-11-21 3 min read # 生活 # 健康 # 牙齿 # 科普 # 读书

很短小的牙齿健康科普书，kindle上看边记笔记不到一个小时就看完了。问答模式的分类合集，基本上涵盖了比较常见的牙病，常识科普辟谣，保养和一些治疗上的问题。有的回答，风趣幽默，打比方的方式让人忍俊不禁，我看的时候有些地方是会笑出声的。科普就应该是接地气的，把专业术语用大白话讲明白的。

《造彩虹的人》
2019-09-23 2 min read # 读书

东野圭吾著，吕灵芝译。北京十月文艺出版社出版。

奇妙的设定——“光乐”，以光为信息传递的媒介。整个故事里看着仿佛是孩子与父母之争，新兴人类危及政治统治而被整治。主人公光瑠演奏光乐，唤醒更多人对于光的感知，启发新的光乐演奏者觉醒的行为像是一个革命者。超自然现象下，其实探讨的是对于不同的包容态度，对于世代更替的必然趋势的肯定，和对于一切妄想阻止时代潮流的人的嘲讽。

想起闻一多先生的那句——“你们杀死一个李公朴，会有千百万个李公朴站起来！”

《大设计》
2019-08-28 5 min read # 读书

史蒂芬·霍金和列那德·蒙洛迪诺著，译者是吴忠超。湖南科技出版社出版。

看过了霍金的《时间简史》《果壳中的宇宙》，现在补上这本。《大设计》物理学理论的东西读起来没那么艰涩，反而感觉谈论哲学的带来的启发更多一些。翻译语言上显得繁琐，不过学术的东西，英文说得很简单，翻译成中文的确是挺难的。要是有时间找找英文原版的话，可以体会一下简洁之美吧~

全书中心思想如译者前言所说，依赖模型的实在论，理解了这个，后面的细节推导倒没那么重要，毕竟于我而已，只是科普消遣，并不是学术研究的目的。
