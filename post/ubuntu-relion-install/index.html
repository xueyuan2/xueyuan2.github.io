ubuntu安装relion

2020-04-08 1 min read # biology # software

1. VMware虚拟机安装(略)

2. Ubuntu配置(略)

3. Relion安装

1. git的下载安装(略)

2. 克隆Relion

git clone https://github.com/3dem/relion.git

3. 配置cmake

  • 下载安装cmake
  • 安装openmpi
    Ubuntu 下安装 openmpi 需要同时安装下面三个包：
sudo apt-get install openmpi-bin openmpi-doc libopenmpi-dev
  • 下载安装fftw和fltk
sudo [apt-get](https://www.baidu.com/s?wd=apt-get&tn=SE_PcZhidaonwhc_ngpagmjz&rsv_dl=gh_pc_zhidao) install fftw3 fftw3-dev fftw3-doc
sudo apt install fltk1.3-dev

4. 安装Relion

cd relion
mkdir build
cd build
make -j <N>
sudo make install

Linux中编译安装软件的基本流程：

1.下载软件
2.解压软件 tar -xzvf 软件包名字＋扩展名
3.编译安装依次使用三个命令

./configure
make
make install
