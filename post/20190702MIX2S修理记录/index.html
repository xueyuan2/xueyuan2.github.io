MIX2S修理记

2019-07-02

手机用了一年多，出现三个问题：

  1. 电池耗电快
  2. 指纹触摸板失效
  3. 屏幕碎裂，显示正常但半个屏幕无法触屏操作

之前碎屏去官方换屏，花费720元。这次为了省钱因为穷, 就自行购置各配件，更换总成带中框，电池，指纹排线模板啦~

1. 带中框总成

  1. 购买：淘宝，270元包邮。
  2. 拆机： MIX2S更换总成视频，优酷和爱奇艺上有。简单~
  3. 试用： 店家是承诺不拆可以退货的，所以先把插口插上，试用，还是跟随视频，完成屏幕显示和触控功能的检测
  4. 装机：验证无误，就可以把原本的模板一一扣下来装上去了~

2. 电池

  1. 购买： 淘宝， 48.02元包邮
  2. 拆机：拆机同上
  3. 试用：这个不好说，插上，充电看看正常就行吧
  4. 装机：电池胶贴好，放进去就好。电池胶比较好拆，电池损耗如果一年后又比较严重，可以方便地拆了再换~

3. 指纹排线

  1. 购买： 淘宝，14.9元包邮
  2. 拆机：拆机同上
  3. 试用： 也是先不做其他，插上插口，试试指纹功能，这个要注意重启一下手机才能识别
  4. 装机：验证无误，可以把旧的扣下了，胶去除干净再用新胶把新的触摸板黏上

总结

1. 自己动手很有意思，出发点是，搞不好也当练手工了~

2. 先找的视频进行学习和了解，之后再购买的配件

3. 对自己的动手能力越来越有信心了呢，XD

修修补补又三年，希望能凑合到博士毕业，5G普及再更换~
