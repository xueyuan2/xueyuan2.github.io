《知乎圆桌·牙齿保卫战》

2019-11-21 3 min read # 生活 # 健康 # 牙齿 # 科普 # 读书

很短小的牙齿健康科普书，kindle上看边记笔记不到一个小时就看完了。问答模式的分类合集，基本上涵盖了比较常见的牙病，常识科普辟谣，保养和一些治疗上的问题。有的回答，风趣幽默，打比方的方式让人忍俊不禁，我看的时候有些地方是会笑出声的。科普就应该是接地气的，把专业术语用大白话讲明白的。

牙病

1. 牙周炎

长期炎症所致，开始可能是细菌所致，但后期基本上是由于免疫细胞过度反应所致

2. 龋齿

四联因素学说

  • 致龋菌

  • 环境

  • 宿主

  • 时间

3. 牙痛

牙神经受压迫所致

常识

1. 洗牙

常规刷牙不能去除的牙结石，可以定期洗牙去除

辟谣：

  • 牙龈萎缩，牙缝看起来变大是不是洗牙的锅？

​ 牙结石积累导致牙龈炎等疾病，洗牙去除填在空隙里的牙结石，导致看起来牙缝大。

  • 洗牙会不会传播疾病？

​ 洗牙时会出血，相关器械一定要消毒杀菌，或者一次性的，去靠谱的地方，这些是可以保证安全的。

2. 牙膏选择

牙膏的主要成分类似，主要作用是摩擦中去除牙齿表面的污垢。

  • 含氟牙膏有防龋的作用，但是注意，不要把牙膏吞下去。

  • 美白牙膏慎用

  • 一般的药物牙膏类似敷料作用，但是含抗生素的牙膏慎用，真有问题，不是牙膏能解决的

  • 不要连续使用相同牙膏，特别是药物牙膏

保养

1. 刷牙的方式

  • BASS刷牙法
  • 刷的再好，还是要定期洗牙，可以预防牙病的发生

2. 刷牙的时间

  • 早饭后刷牙
  • 一日两次

治疗

1. 正畸

  • 病人积极配合
  • 好医生很重要，正统正畸至少本五硕三
  • 正畸本身是科学可行的

2. 智齿

  • 智齿没完全萌出，或者不正，会形成一个口袋，食物残渣引起冠周炎——“牙冠周围那道缝里发炎”，这种“结构缺陷”唯有拔除

  • 不痛不痒的智齿拔不拔，遵医嘱

  • 找正规诊所，好医生

3. 牙龈萎缩

  • 生物学宽度的概念： 牙龈上端距牙齿总是2 mm
  • 牙龈萎缩往往是牙齿出问题，而不是牙龈自身问题
  • 牙龈萎缩没有好的治疗手段，但是对待牙齿问题早发现早治疗，可以避免发生

4. 种牙

  • 看的好痛，保护好牙齿，希望不会走到这一步

总结

保护牙齿，从正确刷牙，定期洗牙开始。
